Projects

CONNECTOR

A Full Stack Social Network Application

  • Designed a full stack social network application where users can update their profiles, connect, leave a comment for a post with MongoDB, Node.js, React, Redux.

  • Developed RESTful APIs by Node.js with extensive API testing with Postman.

  • Manipulated Mongoose to store data on MongoDB, and used JSON Web Token to support authentication.

  • Applied Redux for app state management. Deployed to Heroku with a post-build script via Git as version control.

  • Developed a web application for users to search resources and get recommendation live videos with responsive UI components by using React.js.
